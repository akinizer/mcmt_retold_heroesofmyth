This project is a part of MCMT mod series. It includes improvements on campaign protagonists and aims to help in faster campaign clears. These are what you can find:

* It considers all campaigns from Fall of Trident to Obsidian Mirror
* Increased regeneration
* Increased armor vulnerability
* Increased damages for both hand and ranged attacks
* Multi-hit in small scale for both hand and ranged attacks
* Increased maximum velocity
* Increased number of projectiles for ranged attacks
* Increased maximum range for both hand and ranged attacks for reachability
* non-generic attacks and abilities are included to the changes

* NOTICE: antagonists that can switch sides are excluded due to balance consideration
